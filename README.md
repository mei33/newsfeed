# News fetcher

## Basic usage
1. Clone the repo to your machine: `git clone %url_to_this_repo%`.
2. Install dependencies: `yarn install`.
3. Run dev build: `yarn serve`;

Other commands can be found in `scripts` section in `package.json` file.
